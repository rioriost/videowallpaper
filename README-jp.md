# VideoWallpaper

`VideoWallpaper` は、macOS のデスクトップ壁紙として動画を再生できるメニューバーアプリです。

選択した動画を壁紙の背面でループ再生し、複数ディスプレイ環境でも各画面に表示します。  
通常のアプリウィンドウは表示されず、メニューバーから操作します。

## 特長

- 動画を macOS の壁紙としてループ再生
- メニューバー常駐でシンプルに操作
- 複数ディスプレイに対応
- クリック操作を邪魔しない壁紙背面表示
- 一度選んだ動画を記憶
- ログイン時に自動起動

## 動作環境

- macOS

## インストール

Homebrew でインストールできます。

```sh
brew install --cask rioriost/cask/videowallpaper
```

## 使い方

1. `VideoWallpaper` を起動します。
2. 初回起動時に、壁紙として使いたい動画ファイルを選択します。
3. 選択した動画がデスクトップの背面で再生されます。
4. 動画を変更したい場合は、メニューバーの `Video Wallpaper` から `動画を変更…` を選びます。
5. アプリを終了したい場合は、メニューバーから `終了` を選びます。

## こんなとき

### 動画が再生されない
- 選択した動画ファイルが移動・削除されていないか確認してください。
- 対応しやすい形式として、`mp4` や `mov` などの一般的な動画形式を使用してください。

### 複数ディスプレイで使いたい
- 接続されている各ディスプレイに対して動画を表示します。
- ディスプレイ構成を変更した場合は、自動で再読み込みされます。

### 起動時に自動で使いたい
- アプリはログイン時の自動起動に対応しています。
- 普段使いの壁紙アプリとして常駐利用できます。

## 注意事項

- 動画の長さや解像度によっては、Mac の負荷やバッテリー消費が増える場合があります。
- 著作権や利用条件を確認した上で、使用する動画を選んでください。
